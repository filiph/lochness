Loch Ness

Oh hi! I'm Filip and this is my submission to the Ludum Dare #33 competition. The topic was: You Are The Monster.

If things break (and they will), try refreshing the website in your browser and/or click the Restart button.

In case you're wondering about the engine: this game is written in Dart. It uses a library I'm continually working on (that I call egamebook) that should make it easy for people to make game(book)s like these. Not ready for prime time. The game also uses my fuzzy logic Dart library. The game's code is on github. It's as horrible as you might expect (made during 2 days), but if you're interested in that, you might want to start with world.dart, where most of the game logic is.
